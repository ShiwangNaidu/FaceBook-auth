# FaceBook-auth
An application for Authentication Using FaceBook.
In this project we have used following dependencies.

- [x] node 
- [x] express
- [x] dotenv
- [x] passport
- [x] passport-facebook
- [x] express-session
- [x] ejs
- [x] nodemon
